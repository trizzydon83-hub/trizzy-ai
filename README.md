## üí° About Trizzy AI

Trizzy AI is a WhatsApp bot built with Node.js and the Baileys library. Features include:

- Responds to commands from everyone
- Fetch Zimbabwe news (`!news`)
- Check currency rates (`!rates`)
- Show weather in Zimbabwe (`!weather <city>`)
- Share fun facts & jokes (`!joke`)
- Music search & download links (`!music <song name>`)
- Pair WhatsApp devices (`!pair`)
- Basic commands: `!ping`, `!help`, `!about`
